Security Policy

At Veil AI, we implement rigorous security measures to protect your data and ensure the integrity of our alumni portal. This policy outlines our commitment to maintaining a secure environment.

1. Infrastructure Security

  • Hosting on secure cloud infrastructure with firewall protection and real-time monitoring
  • Use of HTTPS protocol to encrypt all data in transit

2. User Data Protection

  • All passwords are encrypted using industry-standard hashing algorithms
  • User data is isolated and access-controlled by role and purpose
  • We implement database-level security and access logs to detect unauthorized access

3. Authentication

Users must log in using a verified employee ID and password. Multi-factor authentication (MFA) may be enabled for additional security.

4. Security Practices

  • Regular vulnerability scans and code audits
  • Security patches and updates applied routinely
  • Rate-limiting and CAPTCHA mechanisms to prevent brute force attacks

5. Breach Response

In the event of a security breach, affected users will be notified within 72 hours. We maintain an incident response team to investigate and mitigate any threats.

6. User Responsibilities

  • Keep your login credentials confidential
  • Report suspicious activity immediately
  • Avoid using public devices or unsecured networks to access the portal
